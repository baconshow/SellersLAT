# Fontes do Projeto

Coloque seus arquivos de fonte (.woff2, .woff, .ttf) nesta pasta.

Para usar uma nova fonte:
1. Adicione o arquivo aqui.
2. Declare-a no arquivo `src/app/globals.css` usando `@font-face`.
3. Adicione o nome da fonte no `tailwind.config.ts` se desejar usar classes utilitárias como `font-brand`.
